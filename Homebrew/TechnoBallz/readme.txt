--------------------------------------------------
Instructions:
--------------------------------------------------
1. Unzip the archive in this post to a folder on your SD e.g. "tb"
2. Get the technoballz source distribution from http://linux.tlk.fr/games/TecnoballZ/download/
3. Unzip this to a temp directory.
4. Copy the TecnoBallz folder from TecnoBallz-0.9.1\src folder into the folder you made in (1).

You now have
SD CARD
--> TB
--> TecnoBallz

Notes:

Text input does not work (codes and highscores)

This port uses an unreleased build of my SDL Port (which should be able to replace my first release).
There were some bugs in the unicode handling, no MikMod exports and I'm still after the SFX latency issues.
